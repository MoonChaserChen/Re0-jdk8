# AQS
AQS全称 `AbstractQueuedSynchronizer`，此类继承至 `AbstractOwnableSynchronizer`。
AQS通过FIFO队列实现，是Lock、Semaphore、CountDownLatch等实现的基础。
## AbstractOwnableSynchronizer
仅仅是定义了当前独占的线程 `exclusiveOwnerThread` 及相应的Getter、Setter
